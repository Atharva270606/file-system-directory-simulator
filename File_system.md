*****PROJECT TITLE*****
File System Directory Simulator using N-ary Tree


*****PROBLEM DESCRIPTION*****
Operating systems organize files and folders in a hierarchical directory structure. This project simulates a simple directory system where folders can contain multiple subfolders. The simulator allows users to create, delete, search, and display folders, demonstrating how hierarchical data can be managed using a tree structure.


*****DATA STRUCTURES USED*****
N-ary Tree
An N-ary tree is used to represent the directory structure. Each node represents a folder and each folder can have multiple subfolders (child nodes). Tree traversal is used to search and display the directory hierarchy.


*****EXPECTED FEATURES*****
1. Create Folder (mkdir) – Create new directories.
2. Create Subfolder – Add folders inside existing folders.
3. Delete Folder – Remove folders from the directory.
4. Search Folder – Locate a specific folder.
5. Display Directory Structure – Show the hierarchy in tree format.

Example structure:
Root
 ├── Documents
 │    └── Projects
 └── Pictures
